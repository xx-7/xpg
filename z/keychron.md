# Fn功能键切换

- Windows,   长按 Fn+x+l 3秒
- Mac/Linux, 长按 Fn+k+c 3秒
- 功能键模式下F5,F6用来调节灯效亮度, 按键可能失效

# 灯切换

- Fn+右上角灯键: 开关灯
- 右上角灯键: 切换15种亮灯模式
- Fn+F5减亮, Fn+F6增亮

# 电量查看Fn+B

- 全亮: 电量 >=70%
- 第一排灭: 电量 30-70%
- 第二排灭: 电量 0-30%

# 其它

- 恢复出厂设置(蓝牙连接复位): 长按Fn+j+z 3s
- 多设备切换: 按下Fn+1、Fn+2、Fn+3选择之前连接的设备，进行多设备切换
- 有线下取消休眠模式：长按Fn+s+o
